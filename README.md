# 1/f ゆらぎ解析 Webアプリ (RMS・完全フロントエンド)

ブラウザのみで動作する、音声の音量（RMS）変動を元にした1/fゆらぎ（α値）の解析ツールです。

## 特徴
- **サーバー不要**: Web Audio API と JS上の自作FFT関数を使用し、データは外部へ送信されません。
- **ノイズ耐性**: 窓関数（Hanning）の適用、極端な低周波/高周波の除外、DCオフセットの除去を行っています。
- **連続評価**: 二値判定ではなく、回帰直線から導出された $\alpha$ 値に基づく評価を行います。

## GitHub Pages での公開手順

1. **リポジトリの作成**
   - GitHubで新しいリポジトリ（例: `fluctuation-analyzer`）を作成します。
2. **ファイルのアップロード**
   - このディレクトリにある `index.html`, `style.css`, `script.js` をリポジトリにPush（またはブラウザからアップロード）します。
3. **GitHub Pages の有効化**
   - リポジトリの `Settings` タブを開きます。
   - 左側メニューの `Pages` をクリックします。
   - `Source` を `Deploy from a branch` に設定し、Branchを `main` (または `master`) の `/(root)` に設定して `Save` を押します。
4. **アクセス**
   - 数分待つと、上部にURLが表示されます。そのURLにアクセスすればアプリが利用可能です。
